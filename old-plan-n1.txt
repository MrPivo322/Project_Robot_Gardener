Задача: написать код на C++ (для работы с Arduino) для работы 2 шаговых моторчиков в паре, используя библиотеку Servo. 
Задача моторчика N1:
К нему будет закреплена шестерёнка (условно лего) (изображение номер 1), которая будет передвигаться по рельсе (изображение номер 2) на определённое кол-во делений (как условный тестовый запуск)
Задача для моторчика:
-сделать определённое количество шагов между растениями(на начальном этапе это будет фиксированный шаг)
-остановиться и ждать команды
Задача моторчика N2:
после остановки моторчика N1, моторчик N2 должен сделать n кол-во шагов (чтобы сделать поворот на 180 градусов по часовой стрелке). После этого вернуться в исходное положение.
После выполения работы моторчика N2 повторяется цикл работы моторчика N1. Так условные x раз, которые потом можно будет самому указать в коде, либо подключить ультра звуковой датчик расстояния и полагаться на него (фиксировать изменения в дистанции и если изменения происходят - зафиксированно растение и моторчик n1 останавливается)
